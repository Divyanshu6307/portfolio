Extra Activity 2: Analysis of Variance (ANOVA) Study

This activity explores differences in customer satisfaction scores across multiple product lines using one-way ANOVA.

Activity Overview

In this activity, I investigated whether there are statistically significant differences in customer satisfaction scores across four different product lines. The dataset contained satisfaction scores (on a 10-point scale) from 200 customers.

Research Questions:

  • Is there a significant difference in mean satisfaction scores among the product lines?
  • Which specific product lines show statistically significant differences?
  • What are the practical implications of these findings?

Methodology

Data Collection

The dataset was collected through customer surveys over a 3-month period. Each respondent rated their satisfaction with a specific product they purchased.

Variables

  • Independent Variable: Product Line (4 categories: A, B, C, D)
  • Dependent Variable: Satisfaction Score (1-10)
  • Control Variables: Customer demographics, purchase date

Statistical Methods

The analysis followed these steps:

  1. Data cleaning and validation
  2. Descriptive statistics for each group
  3. Assumption checking (normality, homogeneity of variance)
  4. One-way ANOVA test
  5. Post-hoc Tukey HSD test for pairwise comparisons
  6. Effect size calculation using eta-squared
# Python code for ANOVA analysis
import pandas as pd
import scipy.stats as stats
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# Load dataset
data = pd.read_csv('customer_satisfaction.csv')

# One-way ANOVA
f_value, p_value = stats.f_oneway(
    data[data['product'] == 'A']['score'],
    data[data['product'] == 'B']['score'],
    data[data['product'] == 'C']['score'],
    data[data['product'] == 'D']['score']
)

print(f"F-statistic: {f_value:.4f}, p-value: {p_value:.6f}")

# Post-hoc analysis
tukey = pairwise_tukeyhsd(
    endog=data['score'],
    groups=data['product'],
    alpha=0.05
)

print(tukey.summary())

# Effect size calculation
total_mean = data['score'].mean()
ss_between = sum([len(group) * (group.mean() - total_mean)**2 
                for _, group in data.groupby('product')])
ss_total = sum((data['score'] - total_mean)**2)
eta_sq = ss_between / ss_total

print(f"Eta-squared: {eta_sq:.4f}")

Key Findings

  • ANOVA revealed significant differences between groups (F = 12.73, p < 0.001)
  • Eta-squared = 0.24, indicating a large effect size
  • Tukey's HSD showed significant differences between:
    • Product A and Product C (p = 0.002)
    • Product A and Product D (p < 0.001)
    • Product B and Product D (p = 0.018)
  • Product D had the highest mean satisfaction (8.4 ± 1.2)
  • Product A had the lowest mean satisfaction (6.8 ± 1.5)

Conclusion

The analysis revealed statistically significant differences in customer satisfaction across product lines. Product D outperformed others, while Product A showed the lowest satisfaction. These findings suggest that features of Product D should be examined as potential best practices. For Product A, a deeper investigation into customer complaints is recommended to improve satisfaction.

Limitations

This study is limited by its reliance on self-reported satisfaction data. Future research should incorporate objective usage metrics and longitudinal data to validate these findings.
